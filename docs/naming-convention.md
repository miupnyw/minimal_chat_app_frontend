#### Frontend

- Components: PascalCase (MyFileName)
- Utility / helper files: camelCase (myFileName) or kebab-case (my-file-name)
- CSS / SCSS: kebab-case (my-file-name)

#### Backend

- Node.js: camelCase (myFileName) or kebab-case (my-file-name)
- Python: snake_case
- Java / C#: PascalCase for classes

#### Config & Special Files

- Use lowercase + dots: .env, tsconfig.json
